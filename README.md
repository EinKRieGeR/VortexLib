# VortexLib 📚

**VortexLib** — это современная, сверхбыстрая веб-библиотека для управления и чтения локальной коллекции FB2-книг. Проект создан для тех, кто хранит архивы или собственные коллекции электронных книг и хочет иметь удобный доступ к ним через браузер с любого устройства.

## Ключевые особенности

- **Визуальный каталог**: Автоматическое извлечение и отображение обложек напрямую из FB2-файлов, хранящихся внутри ZIP-архивов.
- **Встроенная читалка**:
    - Полноэкранный режим чтения в браузере.
    - Настраиваемые темы (Светлая, Сепия, Темная, Ночная).
    - Кастомизация шрифтов и размера текста.
    - Автоматическое сохранение прогресса чтения для каждой книги.
- **Конвертация на лету**: Возможность скачивания любой книги в форматах **EPUB** или **MOBI** (автоматическая перепаковка на бэкенде через Calibre).
- **Поддержка гигантских архивов**: Использование потоковой декомпрессии (yauzl) позволяет работать с ZIP-архивами размером 2ГБ+ без нагрузки на RAM.
- **Мгновенный поиск**: Полнотекстовый поиск (FTS5) по названиям, авторам и сериям. Поддержка **Unicode** делает поиск по кириллице регистронезависимым.
- **Умный импорт**: Индексация библиотеки через `.inpx` файлы. Настройка размера чанков позволяет оптимизировать скорость импорта под ваше железо.
- **Современный UX**: Дизайн в стиле "Deep Space", иконки, плавные анимации и адаптивность под мобильные устройства.

## Стек технологий

- **Framework**: [Nuxt 4](https://nuxt.com/) (Vue 3, Vite)
- **Runtime**: Node.js / Nitro
- **Database**: SQLite ([Better-SQLite3](https://github.com/WiseLibs/better-sqlite3)) + [Drizzle ORM](https://orm.drizzle.team/)
- **Processing**: 
  - `yauzl` — потоковая работа с ZIP
  - `cheerio` — парсинг структуры FB2/XML
  - `calibre` — серверная конвертация форматов
- **Styling**: Vanilla CSS (Custom Design System)

## Установка и запуск

### Предварительные требования
- **Node.js**: v18.0.0 или выше.
- **Calibre**: необходим для работы функции конвертации (EPUB/MOBI).
  ```bash
  sudo apt-get update && sudo apt-get install calibre -y
  ```

### Быстрый старт
1. **Клонируйте проект**:
   ```bash
   git clone https://github.com/einkrieger/vortexlib.git
   cd vortexlib
   ```

2. **Установите зависимости**:
   ```bash
   npm install
   ```

3. **Запустите сервер**:
   ```bash
   npm run dev
   ```
   Откройте браузер: `http://localhost:4224`

## Настройка и использование

1. **Библиотека**: Зайдите в раздел **Настройки**. Укажите абсолютный путь к папке, содержащей ваши `.inpx` файлы и ZIP-архивы.
2. **Индексация**: Нажмите кнопку «Обновить библиотеку». VortexLib просканирует INPX файлы и создаст базу данных. Это нужно сделать только один раз.
3. **Обложки**: При первом просмотре каталога обложки будут извлекаться из архивов динамически. Они кэшируются браузером для последующего быстрого доступа.
4. **Конвертация**: На странице книги используйте кнопки EPUB или MOBI. Сервер подготовит файл и предложит его для скачивания.
